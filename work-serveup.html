Serve Up!

HTML, CSS, JavaScript

A browser-based game created using JavaScript and DOM manipulation.

It's lunch time on the cruise ship! Player has to serve the correct dishes to the fussy customers within 60 seconds without making more than 3 mistakes. Game is controlled using 'spacebar' and 'enter' keys on the keyboard.
